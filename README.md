# Customer-Analysis-Dashboard
The dashboard provides insights into customer value, transaction patterns, and regional distribution to support data-driven business decisions.
### Project Overview
This project is an interactive Power BI dashboard focused on analyzing customer behavior, profit ability, and sales performance. The dashboard provides insights into customer value, transaction patterns, and regional distribution to support data-driven business decisions.

### Objectives
- Identify high-value and profitable customers
- Analyze customer purchasing behavior and transaction frequency
- Track total sales, profit, and customer distribution
- Understand regional performance and customer segmentation

### Tools Used
- Power BI
- Data Modeling
- Data Visualization
- DAX (for calculations and measures)

### Dataset
The dataset consists of customer transaction data, including sales, profit, quantity ordered, and returns. It supports multi-dimensional analysis across customers, regions, and time
